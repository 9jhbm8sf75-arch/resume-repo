# Nguyễn Vy Thông — Resume / CV Repository

📄 **Bilingual resume** (English & Vietnamese) maintained as a GitHub Pages site.

## Structure

```
résume-repo/
├── index.md        # Landing page (language selector)
├── en.md           # English resume
├── vi.md           # Vietnamese resume
├── _config.yml     # GitHub Pages Jekyll config
├── assets/         # Custom assets
└── source/         # Original LaTeX/Markdown source files
```

## How to use

1. **Fork/clone** this repo
2. Ensure **GitHub Pages** is enabled (Settings → Pages → deploy from `main`)
3. Visit `https://<your-username>.github.io/résume-repo`

## Maintain

To update your resume:
1. Edit `en.md` and/or `vi.md`
2. Push to GitHub — site auto-deploys
3. (Optional) Export to PDF via browser's Print → Save as PDF
